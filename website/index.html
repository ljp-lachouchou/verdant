Verdant

Multi-agent desktop AI assistant with browser automation, skill system, and sub-agent collaboration. Built with Electron + React.

⬇ Download ★ GitHub

Features

🤖

Multi-Agent Collaboration

Delegate tasks to parallel sub-agents via the task tool. Each sub-agent has full tool access including skills.

🌐

Browser Automation

Built-in Playwright browser tool — navigate, click, type, screenshot, fill forms, and run JavaScript.

⚡

Skill System

Markdown-based skills with permission boundaries. Project-level, global, and compatible directory support.

🔧

Vibe Coding

Delegate coding tasks to external CLI tools (Claude Code, Aider, Codex). Automatic screenshot verification.

🧠

Harness Architecture

The model autonomously decides when to delegate, parallelize, and interact. No forced workflow constraints.

💾

Context Management

Token-aware compaction with soft-delete recovery. Atomic SQLite persistence. Crash-safe.
