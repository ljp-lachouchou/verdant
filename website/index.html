Capabilities

Everything you need,
nothing you don't.

A harness that empowers the model — not a workflow that constrains it. The model decides when to delegate, parallelize, and interact.

Multi-Agent Collaboration

Delegate tasks to parallel sub-agents via the task tool. Each sub-agent runs its own loop with full tool access.

Browser Automation

Built-in Playwright — navigate, click, type, screenshot, fill forms, and run JavaScript. Persistent sessions.

Skill System

Markdown-based skills with permission boundaries. Project-level takes precedence over global.

Vibe Coding

Delegate coding to external CLI tools — Claude Code, Aider, Codex. Automatic post-coding verification.

Context Management

Token-aware compaction with soft-delete recovery. Atomic SQLite persistence. Crash-safe by design.

Harness Architecture

The model autonomously decides when to delegate, parallelize, and interact. No forced workflow constraints.

Under the Hood

Built as a Harness,
not a workflow engine.

The harness provides capabilities — tools, memory, persistence, UI — without constraining the model's workflow. The model has final say.
